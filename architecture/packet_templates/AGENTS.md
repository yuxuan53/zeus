# architecture/packet_templates AGENTS

Work packet templates. These define the required structure for different classes of changes to the Zeus codebase.

## File registry

| File | Purpose |
|------|---------|
| `bugfix_packet.md` | Template for bugfix work packets |
| `feature_packet.md` | Template for feature work packets |
| `refactor_packet.md` | Template for refactor work packets |
| `schema_packet.md` | Template for schema/migration work packets |

## Rules

- Every non-trivial change requires a work packet based on one of these templates
- Templates define required sections — do not omit sections, mark them N/A if inapplicable
- Changes to templates are governance changes
