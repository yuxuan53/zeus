# Executive Plan

这是我对 Zeus 下一阶段的总判断：

## 一句话结论

**你已经完成了 foundation；下一步最值钱的不是重做架构，而是把已有信号链、选择链、执行链接进更强的 learning truth。**

## 我建议你优先做的 5 件事

1. **统一概率真相层**
   解决 `shadow_signals` 有 vector、但 canonical fact surface 不完整的问题。

2. **把 calibration 的样本单位改成 decision group**
   停止直接用 pair rows 当成熟度与评估口径。

3. **做 blocked OOS + promotion registry**
   任何新 calibrator / day0 model / fusion policy 都必须可 replay、可比较、可晋升。

4. **把 Day0 变成 learned residual engine**
   它的数据量大、物理约束清晰、而且对交易行为影响大。

5. **把选择与风控数据化**
   family-wise FDR、city correlation、execution microstructure 都应该进入可学习层。

## 最后产物应该长什么样

- 所有决策都有完整 probability trace
- 所有 active model 都有 OOS provenance
- 所有 trades 都能回放出“为什么选了它”
- Day0 不再主要靠 heuristic seam
- risk/correlation/fdr 不再主要靠 hand-tuned rule
