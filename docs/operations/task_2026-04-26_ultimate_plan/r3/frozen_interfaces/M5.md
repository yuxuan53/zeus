## Findings schema

`exchange_reconcile_findings` stores unresolved/reviewed reconciliation findings. An unresolved partial unique index on `(kind, subject_id, context)` makes repeated sweeps idempotent while allowing new findings after operator resolution.

## Write boundary

- Allowed: `exchange_reconcile_findings` rows.
- Allowed: linkable missing `venue_trade_facts` and legal existing command fill events when the exchange trade references a known local command.
- Forbidden: new `venue_commands` rows for exchange-only state.
- Forbidden: live venue submit/cancel/redeem side effects.
- Forbidden: treating stale/error venue reads as proof of absence.
