## Confusion resolved

- Polymarket docs use `apiKey`, `secret`, `passphrase`; Zeus code exposes `WSAuth(api_key, secret, passphrase)` and emits the official subscription payload.
- Polymarket docs require `markets` to be condition IDs, not asset/outcome token IDs.
- Official PING cadence is 10 seconds; M3 default stale gap threshold is 30 seconds (three missed cadence windows).
- Reconnection/message receipt clears transient connected state but does not clear `m5_reconcile_required`; future M5 evidence owns unblock.
