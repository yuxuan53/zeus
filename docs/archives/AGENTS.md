# docs/archives AGENTS

**Historical only — never active authority.** Files here are frozen snapshots of past governance, architecture, and operational documents. They are preserved for provenance and audit trail, not for decision-making.

## Rules

- **DO NOT** reference archive files from active docs (provenance notes are acceptable, but never "go read this archive")
- **DO NOT** modify archive files — they are historical snapshots
- **DO NOT** promote archive content back to active law without explicit governance approval
- Archive files may contain stale phase numbers, outdated invariant IDs, or superseded design decisions — this is expected
